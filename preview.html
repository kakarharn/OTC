Gas Turbine OTC Controller Reset Impact

เมื่อ OTC Controller ถูก Reset
จะส่งผลต่อกำลังผลิตและค่าปรับ Post Event เท่าไร?

ประเมินผลกระทบจาก Gas Turbine OTC Controller Reset ต่อกำลังผลิต ระยะเวลาคืนความพร้อม และค่าปรับ Post Event จาก NCC ตาม Startup Condition ที่เลือก

ประเมินผลกระทบตาม Startup Condition

เลือก Startup Condition เพื่อประเมินกำลังผลิตที่อาจหายไป ระยะเวลาคืนความพร้อม และค่าปรับ Post Event โดยประมาณ

Gas Turbine OTC Controller Reset ส่งผลอย่างไร?

เมื่อ OTC Controller ถูก Reset ระบบต้องใช้เวลาในการ Recovery กลับสู่ระดับพร้อมรับ Load หาก Startup เสร็จก่อนการ Recovery สมบูรณ์ ความสามารถในการรับ Load อาจต่ำกว่าระดับอ้างอิง ส่งผลให้เกิด MW Loss และต้องใช้เวลาเพิ่มเติมก่อนเข้าสู่กระบวนการ Resumption และการคืนค่าความพร้อม
