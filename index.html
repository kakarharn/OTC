Reset Mechanism

Gas Turbine OTC Controller Reset ส่งผลอย่างไร?

เมื่อ OTC Controller ถูก Reset ระบบต้องใช้เวลาในการ Recovery กลับสู่ระดับพร้อมรับ Load หาก Startup เสร็จก่อนการ Recovery สมบูรณ์ ความสามารถในการรับ Load อาจต่ำกว่าระดับอ้างอิง ส่งผลให้เกิด MW Loss และต้องใช้เวลาเพิ่มเติมก่อนเข้าสู่กระบวนการ Resumption และการคืนค่าความพร้อม

Emergency Response Impact

ผลของการตอบสนองด้วย Force TU ทันทีที่พบปัญหา

Force TU Override ลดผลกระทบได้มากแค่ไหน เทียบกับปล่อยให้ Recovery ตามปกติ

เปรียบเทียบผลการประเมินระหว่าง Current Configuration (ไม่ตอบสนอง) และ Force TU Override (มาตรการฉุกเฉินชั่วคราวเมื่อพบว่า OTC ถูก Reset) — เป็น Scenario สำหรับการศึกษาเปรียบเทียบ ควรผ่านการตรวจสอบและอนุมัติเป็น SOP ก่อนนำไปปฏิบัติจริง

Force TU ทำให้ OTC กลับสู่ Reference ภายในไม่กี่วินาที — เร็วกว่า Startup Duration ของทุก Condition มาก จึงไม่มีช่วงเวลาที่ MW ต่ำกว่าที่ NCC สั่งให้เห็นเลย เมื่อไม่มี Post Event เกิดขึ้นจริง ก็ไม่มี Resumption Process และไม่มีค่าปรับ Post Event เลยแม้แต่บาทเดียว ต่างจาก Current Configuration ที่ปล่อยให้ OTC Recovery ตามปกติ ซึ่งจะมีช่วงเวลาที่ MW ต่ำกว่าที่สั่งจริง และถูกบทปรับตามระยะเวลาที่เกิดขึ้น
